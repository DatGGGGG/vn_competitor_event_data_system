# Cach huong dan nguoi khong ky thuat

Neu user khong biet code, hay huong dan ngan gon theo dung thu tu:

1. Tao Google service account
2. Tao JSON key
3. Mo dung URL Socialdata cua team va lay dung slug tu URL
4. Add email do vao Socialdata
5. Chay script mint token
6. Chay callback de lay `usession`
7. Test GraphQL

## Nhung dieu can nhac user

- khong gui private key vao chat
- chi can gui duong dan file JSON
- tren PowerShell phai dung `curl.exe`
- token Google thuong song khoang 1 gio
- `usession` cua Socialdata theo tai lieu thuong song khoang 24 gio
- slug cua moi user/team co the khac nhau, khong duoc mac dinh `srcvn`
